Privacy Policy

ContextLens Chrome Extension — Last updated June 2026

ContextLens is a Chrome extension that lets you highlight text or select a screenshot region on a webpage, receive an AI-generated explanation, and review saved explanations as history and flashcards. This policy explains what data is handled and how.

Data stored on your device

The following data may be stored locally in your browser using Chrome's built-in storage API (chrome.storage.local):

  • Saved highlights — selected text, optional notes, source page URL/title, AI explanations, follow-up chats, and timestamps
  • Web browsing activity for saved items — the URL and title of pages where you choose to save a highlight or screenshot
  • Screenshots — screenshot regions you choose to save, plus smaller preview images for faster dashboard rendering
  • Flashcard data — sets, review state, due dates, ratings, and related scheduling metadata
  • Account data — your email address and authentication token when you sign in
  • Preferences — theme, accent color, mode, font size, save trigger settings, and screenshot trigger settings

Data sent to ContextLens

When you ask ContextLens to explain or save something, the relevant selected text, screenshot region, optional note, source URL/title for that selected item, chat context, and generated response may be sent over HTTPS to the ContextLens backend hosted on Railway. If you are signed in, saved highlights, screenshots, explanations, flashcard sets, and review metadata are stored in the ContextLens backend so they can sync across your devices.

ContextLens may also store first-party operational metrics such as feature event counts, success or failure status, coarse input type, model/provider used, and response latency. These metrics do not include raw highlighted text, screenshot image contents, notes, or full page browsing history, and are used only to monitor reliability, usage, and costs during beta.

Account data

When you create an account or sign in, your email address and password are sent over HTTPS to the ContextLens backend for authentication. The extension stores your email address and authentication token locally so it can keep you signed in. Passwords are not stored in the extension.

Data shared with service providers

ContextLens uses service providers only to provide the extension's core explanation, screenshot, account, and sync features. Data is not sold and is not used for advertising.

  • Railway / ContextLens backend — hosts the backend API and database used for accounts, sync, saved captures, screenshots, explanations, flashcard sets, and review metadata
  • Google Gemini — may receive highlighted text, screenshot text/image content, notes, and chat context to generate standard explanations
  • Google Cloud Vision — may receive screenshot image content to extract text from screenshots
  • Google Fonts — may be loaded to render Arabic text cleanly in the extension interface
  • Anthropic Claude — may receive highlighted text, screenshot content, notes, and chat context for Deep Dive explanations
  • QuranCDN — when you select text on Quran.com, ContextLens may request verse text by verse key to preserve the correct Arabic source text

Screenshots

When you use the screenshot feature, the visible area of the current tab is captured so you can select a region. Only the selected region is sent for explanation. If you save the explanation or are signed in, the selected screenshot region and a smaller preview may be stored locally and in the ContextLens backend for history, review, and flashcards.

Permissions

The extension requests the following Chrome permissions:

  • storage — to save your highlights and settings locally
  • unlimitedStorage — to support saved screenshots and larger study histories
  • tabs — to identify the active tab and capture the visible tab when you start a screenshot
  • scripting — to inject or refresh the ContextLens content script when needed
  • contextMenus — to add an "Ask ContextLens" right-click option for selected text
  • host permissions (<all_urls>) — to show the selection bubble and screenshot button on webpages where you choose to use ContextLens

Limited use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements. ContextLens uses user data only to provide or improve its explanation, screenshot, history, sync, and flashcard features.

Data retention and deletion

You can delete individual saved items from the ContextLens dashboard. Deleting a synced item removes it from local storage and requests deletion from the ContextLens backend. You can also delete your account from the dashboard or contact us for data deletion requests.

Children's privacy

This extension does not knowingly collect data from anyone, including children.
