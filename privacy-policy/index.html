Privacy Policy

ContextLens Chrome Extension — Last updated May 2026

ContextLens is a Chrome extension that lets you highlight text on any webpage and receive an AI-generated explanation. This policy explains what data is handled and how.

Data stored on your device

The following data is stored locally in your browser using Chrome's built-in storage API (chrome.storage.local). It never leaves your device except as described below.

  • Your highlights — the text you save and any context you add
  • Your Anthropic API key — entered by you in Settings
  • Your preferences — save trigger settings (bubble, context menu, screenshot shortcuts)

Data sent to third parties

When you save a highlight or screenshot, the selected text and your optional context note are sent directly to the Anthropic API using your own API key. This is the only external request the extension makes.

  • Anthropic's privacy policy applies to data sent to their API: anthropic.com/privacy
  • No other third party receives any data from this extension

Data we collect

None. The developer of ContextLens does not collect, receive, store, or have access to any data you enter into the extension. There is no analytics, no telemetry, no crash reporting, and no backend server.

Screenshots

When you use the screenshot feature, the visible area of your current tab is captured temporarily in memory to let you select a region. The selected region is then sent to the Anthropic API for explanation. No screenshot data is stored permanently on your device or transmitted anywhere other than Anthropic.

Permissions

The extension requests the following Chrome permissions:

  • storage — to save your highlights and settings locally
  • tabs — to capture the visible tab for screenshots
  • contextMenus — to add a "Save to ContextLens" right-click option
  • host_permissions (<all_urls>) — to inject the save bubble on any webpage you visit

Children's privacy

This extension does not knowingly collect data from anyone, including children.
